# 🎵 天音かなた 歌アーカイブ | Kanata Songs

ホロライブ4期生 天音かなたの3D Live・歌枠・MVをまとめたウェブサイトです。

## 特徴

- 🎤 3D Live、歌枠、MVをカテゴリー別に閲覧
- 🔍 フィルター機能で簡単に検索
- 📱 レスポンシブデザイン対応
- ⚡ Vite + Reactによる高速な動作

## 技術スタック

- **フレームワーク**: React 19
- **ビルドツール**: Vite 6
- **言語**: JavaScript (JSX)

## セットアップ

```bash
# 依存関係のインストール
npm install

# 開発サーバーの起動
npm run dev

# 本番ビルド
npm run build

# プレビュー
npm run preview
```

## 開発

開発サーバーは `http://localhost:5173` で起動します。

## ライセンス

このプロジェクトはファンメイドのアーカイブサイトです。
すべてのコンテンツの権利は元の著作権者に帰属します。
